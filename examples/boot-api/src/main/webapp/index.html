Access Book Library API step by step:

step1:
  • List first page books(page size=10)
  • List second page books(page size=2)
step2:
  • Login as Calvin, fetch the token
  • Calvin apply the 3rd book which is owned by David, replace the token
  • List the books status again
step3:
  • Login as David, fetch the token
  • David confirm the book borrowed, replace the token
  • List the books status again
step4:
  • David mark the book returned,other one can apply it again, replace the token
  • List the books status again
other flow:
  • Calvin cancel the apply, replace the token
  • David reject the apply, replace the token

Access below management endpoint:

  • http://localhost:7002/health
  • http://localhost:7002/info
  • http://localhost:7002/dump
  • http://localhost:7002/metrics
  • http://localhost:7002/env
  • shutdown(disable by default, POST method)

JMX expose as Restful by jolokia. e.g. Tomcat's MBean:

  • http://localhost:7002/jolokia/read/Tomcat:type=Connector,port=8080

H2 Console in development profile:

  • http://localhost:8080/h2/
